# ClassicGUI

ClassicGUI is a small AutoHotkey v2.0 project. This was available on my googledrive but i am now uploading it to github


## 🛠️ Built with
- AutoHotkey v2
- Notepad (yes, really)
- An addiction of creation


## 🧰 Features
- Awesome AutoHotkey v2.0 GUI
- Start Screen
- Uses current Username of user with A_Username
- Easy to modify and expand
- More features waiting within the actual GUI!
- Macro Creator
- Custom draggable title bar
- With a custom X button


## 📜 How to Run
1. Install AutoHotkey v2 from [the official AutoHotkey page](https://www.autohotkey.com/), https://www.autohotkey.com/ if you dont trust masked links
2. Download ClassicGUI through the ClassicGUIInstaller, **or** just install the .ahk it has a feature to auto set things up for you
3. Run the program, if you are going to use features as such as the lagswitch toggling firewall rules is admin only, so run as administrator

Then boom! you have an awesome interactive GUI with many features such as a basic macro creator!


## 💻 More about ClassicGUI
I made this project as my first ever big project and i wanted to do it with autohotkey because of how awesome it was, and its not supposed to be a cheat client or whatever, thats against many games TOS but its more of a multi-tool GUI

This was made for roblox if you couldnt tell but has many features that arent limited to Roblox as for one of it's features, it creates a firewall rule for the lagswitch. However the lagswitch doesnt have to only be for the roblox client, you can rebind the game it targets to anything else with setting the program it targets.
