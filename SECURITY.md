# Security

This project is fully open‑source. You should never receive malware from this project as long as you download it from the official repository:

https://github.com/SonicSpeed13425/ClassicGUI.ahk

**Or** from the user https://github.com/SonicSpeed13425

ClassicGUI uses `A_Username` to display your Windows username on the start screen. This is expected behaviour and not a security risk.
